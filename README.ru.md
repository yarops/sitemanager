Site Manager
===================================

Веб-сервис для управления сайтами с функцией мониторинга.

## Возможности

- Управление сайтами и мониторинг
- Управление серверами
- Система шаблонов
- Контроль доступа пользователей
- Проверки доступности
- Обработка через очереди

## Установка

1. Склонировать репозиторий.

2. Выполнить установку ```composer install``` или обновление зависимостей ```composer update```.

3. Выполнить команду и следовать инструкциям настроить окружение dev или prod:
  ```
  php /path/to/yii-application/init
  ```

4. Настроить подключение к БД в конфиге:
  ```
  /common/config/main-local.php
  ```

5. Запустить миграции:
  ```
  php yii migrate
  ```
